## Quick Start
---

```bash
git clone https://github.com/eDavila-DrRaccoon/vision_pipeline.git
cd vision_pipeline
docker compose up --build
```

The project will:

- download the YOLO11 model (first run only)
- perform inference
- save the annotated image
