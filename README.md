## Quick Start
---

```bash
git clone https://github.com/eDavila-DrRaccoon/vision_pipeline.git
cd vision_pipeline
docker compose up --build
```

The project will:

- build the Docker image
- start the FastAPI service
- download the YOLO11 model (after the first run/request only)
- perform object detection after the request is made to the API endpoint
- save the annotated image to the configured output directory (default: `outputs/predict/`)

## REST API
---

Start the service:
```bash
docker compose up --build
```

Open:
```
http://localhost:8000/docs
```

Interactive API documentation is automatically generated by FastAPI and allows testing every endpoint directly from the browser.

### Example request:
```json
{
  "image": "examples/images/dog_and_person.jpg"
}
```

### Example using curl
```bash
curl -X 'POST' \
  'http://127.0.0.1:8000/inference' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "image": "examples/images/dog_and_person.jpg"
}'
```

### Example response:
```json
{
  "status": "success",
  "message": "Inference completed successfully.",
  "data": {
    "output_directory": "outputs/predict"
  }
}
```

### Example error
```json
{
  "detail": "Image not found: examples/images/dog_an_person.jpg"
}
```
